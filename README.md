# KeyTop Fresh - Sprint 1 Project Submission

KeyTop Fresh is a responsive multi-page website developed for a juice, fruit salad, and ice cream shop. The project was implemented using HTML, CSS, and JavaScript while meeting all academic requirements for structure, design, and functionality.

# Project Summary

The website includes the following pages:
- Home
- Menu
- Order
- About
- Admin (Order Management)

Users can browse products, add items to a cart, modify quantities, and complete an order using a validated customer form.

# Technical Implementation - Semantic HTML

The project uses proper semantic elements including:
- <header>
- <nav>
- <main>
- <section>
- <article>
- <aside>
- <footer>

# CSS & Responsive Design

The layout is responsive and adapts to different screen sizes.
CSS variables are used for consistent branding and design structure.

# JavaScript & DOM Manipulation

The project includes dynamic features such as:
- Add to cart functionality
- Cart counter updates
- Quantity modification
- Item removal
- Automatic subtotal, tax (15%), and total calculation
- Order validation
- Popup confirmation messages
- Customer review carousel
- Invoice generation with auto-incrementing invoice number

# localStorage Usage

The application uses localStorage to:
- Store cart items
- Save completed orders
- Generate unique invoice numbers
- Display saved orders in the Admin page

# Order & Invoice System

When a customer completes an order:
- Customer details are validated
- Cart items are saved
- A unique invoice number is generated (KT000100 format)
- Subtotal, tax, and total are calculated
- An invoice is displayed
- The cart is cleared

# Conclusion

KeyTop Fresh fulfills all project requirements including semantic structure, DOM manipulation, localStorage integration, responsive design, and GitHub submission.
The system demonstrates structured JavaScript logic, organized layout design, and clean user interaction flow.
